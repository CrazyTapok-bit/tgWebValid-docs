If you are sure that the verification data is correct, but you still keep getting this exception, please check the correctness of the token specified in the settings.

BotException

The name of the bot you are trying to apply is incorrect, or the bot does not exist in the added list.

ValidationException

Failed to validate the user with one of the available validators. It can potentially indicate a threat of hacking and/or bypassing the system.
